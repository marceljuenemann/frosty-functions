Cross-chain signing

Frosty Functions can sign arbitrary messages with threshold-secured keys, enabling trust-minimized interaction across all EVM chains ‚Äî with Bitcoin support coming soon. No central private key, no off-chain signer to trust.

Executed on-chain

Functions run on a subnet of the Internet Computer: a byzantine-fault-tolerant blockchain operated by 13 independent node providers worldwide. Execution is verifiable, replicated, and not controlled by any single party.

EVM interoperability

Invoke Frosty Functions easily from smart contracts or EOAs via the bridge contract. Pay gas in ETH, receive signed transactions, and write your logic in AssemblyScript with familiar TypeScript-like syntax.
